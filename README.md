Local Setup:
```bash
git clone https://github.com/nickmwangemi/cityapi.git
cd cityapi
pipenv shell
pipenv install
hypercorn main:app --reload
```

Crack open the API docs (powered by OpenAPI) at [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)
